Are the Html tags and element the same thing?

 HTML Tags and Elements are sometimes perceived as the same. However, they are not. There is a subtle difference between HTML elements and tags that many people aren't aware of. HTML Tags are building blocks of HTML Page. They tell the browser how it should display content to the user. A tag starts with a < bracket and ends with a > bracket.

What is the 'class' attribute in HTML?With Example.

 The class attribute in HTML is used to define a class of elements. It is used to group elements with similar properties. The class attribute can be added to any HTML element. For example, if you want to style all the headings in your document with a specific font and color, you can add the class attribute to all the headings and then define the style for that class in your CSS file.

What is the differant between a blocl-level element and inline element?

 A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.
An inline element does not start on a new line.
